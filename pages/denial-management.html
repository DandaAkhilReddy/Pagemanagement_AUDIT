Detailed Resolution Steps by Category

Medical Necessity Denials

Resolution Process:

  • Review documentation for missing elements
  • Obtain additional records from provider
  • Prepare appeal letter with clinical justification
  • Submit within payer's appeal timeframe
  • Track appeal status weekly
Prior Authorization Denials

Resolution Process:

  • Verify if hospital obtained authorization
  • Request retro-authorization if possible
  • Submit clinical documentation
  • Escalate to medical director if needed
  • Document authorization number
Coding Error Denials

Resolution Process:

  • Identify specific coding issue
  • Review documentation for support
  • Correct CPT/modifier/diagnosis
  • Resubmit corrected claim
  • Update provider education log

Provider-Specific Denial Patterns

High-Risk Providers Requiring Training

Common Provider Issues:

  • Missing Q6 modifier: Locum tenens not indicated
  • Insufficient documentation: Level of service not supported
  • Wrong place of service: Inpatient vs observation
  • Bundled services: Billing separately when should bundle
  • Time-based coding errors: Critical care documentation
